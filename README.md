# [Release] Searcher ULP – Fast Folder Text Search & Combo Extractor (by dEEpEst – HTDark)
I'm sharing a free tool I recently released: **Searcher ULP**, a GUI-based folder text search engine designed for large files and working without the interface freezing.

## 🔑 Main features
- 📂 Searches thousands of files and subfolders instantly.  
- 🔎 Supports patterns (e.g., `.txt; .log; .json; .xml; *.conf`, and many more).  
- ⚡ Streaming scanning of large files (no crashes).  
- 🧩 Automatic detection and extraction of `user:pass` combos.  
- 📤 Exports results to CSV/TXT, with deduplication and normalization options.  
- 🖥️ Lightweight and responsive GUI (with DPI support on Windows).
